# Script_Html_CsS

Collection of small front-end demos in HTML, CSS, JavaScript, and jQuery.

## Contents

This repository contains several UI examples and effects, including:

- animated buttons
- modal windows
- slideshows
- hover effects
- search bars
- login forms
- image effects

The `apercus/` folder contains preview images for the demos.

## Project structure

Each demo is stored in its own folder and can usually be opened directly in a browser from its `index.htm` or `index-1.htm` file.

Examples:

- `slideshow/`
- `barre-recherche-depliante/`
- `fenetre-modale/`
- `image-animation-accordeon/`

## Usage

1. Open one of the demo folders.
2. Launch the corresponding HTML file in your browser.
3. Explore the related `css/`, `js/`, and asset files.

## Notes

This repository is intended as a gallery of front-end experiments and reusable interface snippets.
